一，JSX语法简介 #

JSX就是Javascript和XML结合的一种格式。React发明了JSX，可以方便的利用HTML语法来创建虚拟DOM，当遇到<，JSX就当作HTML解析，遇到{就当JavaScript解析。 例如：

javascript
<ul className="list-class">
    <li>list-item1</li> 
		<li>list-item2</li>
</ul>

代码中可以看到 jsx中使用样式class 需要使用 className 注意的是，自定义的组件必须首写字母要进行大写，而JSX是小写字母开头的。 一个简单的jsx语法进行三目运算:

javascript
import React from 'react'
const Component = React.Component

class App extends Component{
    render(){
        return (
            <ul className="list-class">
                <li>{true?"我是list1":"我不是，别乱说"}</li> 
                <li>list-item2</li>
            </ul>
        )
    }
}

export default App;

二，JSX语法实操 #

代码todolist：

javascript
import React,{Component} from 'react'

class Elample1 extends Component{
    render(){
        return  (
            <div>
               <div><input /> <button> 添加todo </button></div>
               <ul>
                   <li>今天要写代码</li>
                   <li>今天要吃饭</li>
               </ul> 
            </div>
        )
    }
}
export default Elample1

注意：编写jsx的时候 return返回的html中需要包裹一个最外层的div或者html元素，这就跟vue的template类似，组件中不用一个最外层元素包裹 就会报错 有的时候，我们并不想用html元素去包裹代码，这个时候 就可以使用react为我们提供的Fragment标签去进行包裹 代码：

javascript
import React,{Component,Fragment } from 'react'

class Elample1 extends Component{
    render(){
        return  (
           <Fragment>
               <div><input /> <button> 添加todo </button></div>
               <ul>
                   <li>今天要写代码</li>
                   <li>今天要吃饭</li>
               </ul> 
            </Fragment>
        )
    }
}
export default Elample1

三，JSX 中的注意事项 #

  1. jsx中给节点起类名，我们之前是用class 就可以了，jsx中虽然现在页面是可以正常显示结果的，但是你代开浏览器控制台会发现Warning警告。

要把class换成className，防止与js中的class类名 冲突

  1. 在vue中，我们知道渲染后端返回的富文本字段可以使用v-html，但是在react中，其实提供了一个属性 dangerouslySetInnerHTML
javascript
<div dangerouslySetInnerHTML={{__html: '<a href="">跳转</a>'}}></div>
//若是动态数据直接替换__html的值即可。
<div dangerouslySetInnerHTML={{__html: xxx字段}}></div>
  1. JSX中的<label>，label是html中的一个辅助标签，也是非常有用的一个标签。在react中我们需要 htmlFor 这样来使用
javascript
<div>
    <label htmlFor="label">增加todo：</label>
    <input id="label" className="input" value={this.state.inputValue} onChange={this.inputChange.bind(this)} />
    <button onClick={this.addList.bind(this)}> 增加 </button>
</div>
